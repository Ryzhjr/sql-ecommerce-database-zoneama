# Projet SQL - Base de données e-commerce ZoneAma

## Description

Ce projet consiste à concevoir une base de données relationnelle pour un site e-commerce fictif nommé ZoneAma.

L’objectif est de gérer :
- les produits fabriqués ;
- les produits approvisionnés ;
- les fournisseurs ;
- les commandes ;
- les lignes de commande ;
- les relations entre fournisseurs et produits.

Le projet comprend la modélisation de la base de données, la création des tables SQL, l’ajout de contraintes, l’insertion de données de test et l’exploration de la base à travers plusieurs requêtes.

## Technologies utilisées

- SQL
- MySQL
- Looping pour la modélisation conceptuelle
- GitHub pour la documentation et le versioning
